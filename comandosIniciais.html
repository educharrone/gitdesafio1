Comando Objetivo
git init Cria um repositório Git vazio ou reinicializa um já existente
git add . Adiciona o conteúdo do arquivo ao índice / stage
git commit -m "texto descritivo" Grava alterações no repositório
git branch -M "main" Move para branch main
git remote add origin git@gitexemplo Associa o repositorio local ao repositorio remoto indicado
git push -u origin main Atualiza refs remotos junto com objetos associados
